▪ Current Release Highlights

Update V0.1

This release features a fully functional equip/skip aura loop alongside the following standard deployment systems: Roll, Autoroll, and Quick roll.

System Name Behavior & Description
Roll System Pressing the Roll button triggers a short animation of approximately 2.5s or 3.2s. Once the animation finishes, your rolled aura is displayed with clean options to either equip or skip the result.
Autoroll System Keeps rolling continuously and automatically for your account without requiring manual input.
Quick Roll System Bypasses all visual effects and roll animations to render results instantly for accelerated testing.
